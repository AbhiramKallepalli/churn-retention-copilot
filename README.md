# AI Decision Copilot for Customer Retention

A local LLM-powered agentic decision-support tool that helps business 
analysts identify high-risk customers and recommend retention actions.

## Project Overview
- **Tool:** Customer churn risk analyzer with LLM-driven explanations
- **Model:** Phi-3 Mini (runs locally via Ollama — no cloud required)
- **Dataset:** IBM Telco Customer Churn (Kaggle)
- **Interface:** Command-line (CLI)

## Repository Structure
- `/data` — dataset files (CSV)
- `/src` — Python source code
- `/tests` — unit tests
- `/outputs` — session logs and results
- `/docs` — project specification and documentation
